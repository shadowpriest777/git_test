React Advanced

Видеокурс React Advanced является продолжением курса React Essential, и в этом курсе вы получите возможность изучить React детальнее и глубже.

Данный курс научит вас понимать, как настраивать окружение для будущих проектов, писать полноценные приложения на Redux, используя различные инструменты, которые делают разработку удобнее и эффективнее.

Angular 2

Angular 2 — очень мощный фреймворк для создания масштабируемых веб-приложений, которые работают в режиме реального времени без перезагрузки страницы. С возможностями TypeScript Angular стал силен как никогда!

На ITVDN вы можете пройти обучение по курсу Дмитрия Охрименко Angular 2 Essential. Курс состоит из 7 уроков.

Больше информации о возможностях Angular 2 и особенностях работы с ним вы почерпнете из первого урока нового курса Дмитрия Охрименко.

PHP Starter

На ITVDN уже доступны 11 уроков нового курса PHP Starter.

Полный курс будет состоять из 16 уроков.

В курсе содержатся базовые понятия языка PHP, которые включают в себя синтаксис и использование различных функций для работы. После прохождения курса Вы сможете понимать синтаксис и принцип работы языка, писать и использовать функции, обрабатывать строки, массивы, переменные.
